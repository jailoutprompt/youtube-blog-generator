Risks & Dependencies
리스크 + 의존성
! OpenAI API 비용
유저가 많아지면 API 비용이 수익을 초과할 수 있음. gpt-4o는 건당 ~₩50~200.
대응: 무료는 mini만, 프로 마진 계산 필수. 캐싱으로 중복 절약.
! yt-dlp 차단 위험
YouTube가 yt-dlp를 차단하면 자막 추출 불가. 핵심 기능이 깨짐.
대응: YouTube Data API v3 백업 경로 준비. 자막만이면 API로 충분.
! 경쟁 서비스
Jasper, Writesonic 등 기존 AI 글쓰기 도구. YouTube 특화는 적지만 진입 가능.
대응: "YouTube 전문" 포지셔닝. 채널 일괄 + 전자책은 차별점.
! SQLite 한계
동시 접속 많아지면 SQLite 락 이슈. 프로토타입에서 넘어가야 함.
대응: 유저 100명 넘으면 PostgreSQL 전환. Prisma ORM 도입.
! API 키 보안
OPENAI_API_KEY 유출 시 대규모 과금. 서버 사이드만 호출 중이라 현재는 OK.
대응: 환경변수 관리 철저. 배포 시 시크릿 매니저 사용.
! Whisper 서버 부하
로컬 Whisper는 CPU 집약적. 동시 요청 시 서버 과부하.
대응: OpenAI Whisper API로 전환 (유료). 또는 큐 시스템 도입.
